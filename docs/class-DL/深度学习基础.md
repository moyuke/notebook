# 深度学习基础

!!!tip "深度学习 = 人工神经网络 + 训练学习基础"

## 人工神经网络

- 多层感知机 MLP
- 卷积神经网络 CNN
- 循环神经网络 RNN
- Transformer
- 图网络 GNN

## 数据

* 一维：文本、声音
* 二维：图像
* 三维：点云视频

### 图像

* 每个网格用一个像素表示，规则有序

* $ I \in R^{H\times W\times C} $

  * H：height，W：width

  * C：channel通道数，RGB=3

### 文本

* 构建词典
* 转化为数字序列(1,2,3)，再转化为vector([1,0,0],[0,1,0],[0,0,1])
* word embedding转化
